Application pour la fiche Erasmus

Cette application web a pour but de vous aider à remplir votre feuille Erasmus+ qui permet l'obtention de la bourse Erasmus+. Cette bourse est disponible à partir de la Licence 2. Vous pouvez également utiliser cet outil pour visualiser quels cours prendre ce semestre. A noter que certains cours sont à choisir parmi une sélection (signalé dans la colonne "Notes"). Pour des raisons de sécurité, les données ne sont pas stockées et tous les champs sont réinitialisés si vous quittez le site.

Pour des raisons d'heures de travail, le site est bien moche voire inutilisable sur téléphone :D
